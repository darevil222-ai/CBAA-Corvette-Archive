## Import Instructions
1. Backup your save.
2. Import JSONs using your preferred NMS save editor.
3. Spectre GOF2HD - Construction Data.json - this data goes to BaseContext.PlayerStateData.PersistentPlayerBases[xx] - you need to find your Corvette build that you want to replace (this folder also contains your planetary bases - so be careful)
4. Spectre GOF2HD - ship data.json goes to BaseContext.PlayerStateData.ShipOwnership[yy] - Y can be from 0 to 11 (12 slots), and you will be replacing the same corvette that you changed data in the previous point.
